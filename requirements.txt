# On university HPC clusters CuPy is often available as a module:
#   module load cupy
#
# Verify the installation:
#   python -c "import cupy; cupy.array([1.0]); print('CuPy OK')"
#
# If CuPy is not installed the app starts normally in CPU-only mode.
